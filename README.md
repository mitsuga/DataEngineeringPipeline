# Data Engineering Pipeline

## Project Overview
This project demonstrates a simple Data Engineering Pipeline using Python and Pandas. It reads employee data from a CSV file, cleans missing values, generates a report, and saves the cleaned data.

## Features
- Read employee data from a CSV file
- Detect missing values
- Fill missing Age and Salary values using the average
- Save cleaned data to a new CSV file
- Generate a project report automatically

## Technologies Used
- Python
- Pandas

## Project Files
- employees.csv - Input dataset
- pipeline.py - Main Python program
- cleaned_employees.csv - Cleaned dataset
- report.txt - Generated report
- requirements.txt - Required Python library

## How to Run
1. Install the required library:
   pip install pandas

2. Run the program:
   python pipeline.py

## Output
- cleaned_employees.csv
- report.txt
